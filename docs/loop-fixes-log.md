# Лог автономных правок (/loop)

Журнал мелких исправлений логики и багов, найденных в автономном режиме `/loop`.
Каждая итерация — одна правка с обоснованием. Один общий файл на все операции.

---

## Итерация 1 — 2026-06-10

**Файл:** `src/components/ProductCard.tsx`
**Тип:** DRY / дублирование логики

**Проблема:** В `ProductCard` минимальная цена вычислялась вручную
(`Object.values(perfume.prices).filter(...).then Math.min`) — точная копия тела
`getMinPrice()`, которую ввели в этапе 1 миграции именно ради устранения таких
дублей. Остальные потребители (`CatalogClient`, `product/[id]`) уже использовали
`getMinPrice`, а `ProductCard` остался с копипастой.

**Исправление:** заменил ручной расчёт на `getMinPrice(perfume)` из `@/data/utils`.
Поведение идентично (та же фильтрация `> 0` и null при отсутствии цен), но логика
теперь в одном месте — при изменении правила «минимальной цены» не разъедется.

**Проверка:** `npm run build` — зелёный, 16/16 страниц, типы валидны.
