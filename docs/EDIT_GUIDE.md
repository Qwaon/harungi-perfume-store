## 2. Шапка сайта (Header)

**Файл:** `src/components/Header.tsx`

### Логотип / название

```tsx
// строка ~35
<Link href="/">
  HARUNGI   {/* ← сюда вписать новое название */}
</Link>
```

### Пункты навигации

```tsx
// строки ~17–21
const navLinks = [
  { href: '/catalog', label: 'Каталог' },   // ← текст пункта меню
  { href: '/about',   label: 'О нас'    },
  { href: '/faq',     label: 'FAQ'       },

```

Чтобы добавить новый пункт — добавьте ещё одну строку в этот массив.  
Чтобы убрать — удалите соответствующую строку.

### Кнопка "Заказать" в шапке

```tsx
// строка ~51
<Link href="/catalog" className="btn-primary ...">
  Заказать   {/* ← текст кнопки */}
</Link>
```

## 3. Главный экран (Hero)

**Файл:** `src/components/Hero.tsx`

### Фоновая фотография

```tsx
// строка ~20
<img
  src="https://images.unsplash.com/photo-1541643600914-78b084683601?w=1920&q=80"
  //   ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
  //   Замените на URL своей фотографии
```

> **Как добавить свою фотографию:**
> 1. Загрузите фото в папку `public/` (создайте если нет)
> 2. Укажите путь: `src="/hero-bg.jpg"` (если файл называется `hero-bg.jpg`)
> 3. Рекомендуемый размер: 1920×1080 px или шире, тёмные тона

### Подзаголовок-лейбл (маленький текст над заголовком)

```tsx
// строка ~33
Нишевая · Оригиналы · Распивы
// ↑ Замените на нужный текст
```

### Главный заголовок (H1)

```tsx
// строки ~40–43
Не соглашайтесь<br />
на <span className="italic ...">меньшее</span>
// ↑ Замените на свой заголовок. Тег <br /> — перенос строки.
// Слово в <span className="italic ..."> будет курсивным.
```

### Описание под заголовком

```tsx
// строка ~51
Оригинальная нишевая парфюмерия. Распивы от 2 мл — пробуйте прежде, чем вкладывать в полный флакон.

```

```tsx
// строки ~68–72
{ value: '50+',   label: 'Ароматов' },

{ value: 'от 2 мл', label: 'Распивы' },
// ↑ Меняйте цифры и подписи под ними
```

## 10. Каталог и карточки товаров

### Источник данных о товарах

**Файл:** `src/data/perfumes.json`

Это главный файл с товарами. **Не редактировать вручную** — при следующей синхронизации с Airtable перезапишется. Управление ведётся через Airtable CMS (см. `docs/airtable-setup.md`).

Чтобы вручную изменить товар (временно):

```json
{
  "id": "dior-sauvage",
  "name": "Sauvage",
  "brand": "Christian Dior",
  "description": "Ваше описание аромата...",
  "prices": { "2ml": 400, "5ml": 900, "10ml": 1600 },
  "featured": true,
  "images": ["https://...url-фотографии..."]
}
```

### Как поменять фото товара

В каждом товаре есть поле `"images"` — массив URL фотографий.

```json
"images": [
  "https://images.unsplash.com/photo-xxx",
  "https://images.unsplash.com/photo-yyy"
]
```

- **Первое изображение** — показывается в каталоге и как основное на странице товара.
- Можно добавить 2–4 изображения — они станут галереей на странице товара.
- Используйте прямые URL (Unsplash, ваш хостинг изображений и т.д.)

### Какие товары попадают на главную страницу

На главную выводятся только товары с `"featured": true`.

```json
"featured": true   ← показывается на главной
"featured": false  ← только в каталоге
```

## 13. SEO и мета-теги

**Файл:** `src/app/layout.tsx` — глобальные мета-теги для всего сайта.

```tsx
export const metadata = {
  title: {
    default: 'HARUNGI — Нишевая парфюмерия',
    //        ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ ← название вкладки браузера
  },
  description: 'Оригинальная нишевая и селективная парфюмерия...',
  //             ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
  //             ← описание для Google (160 символов максимум)
};
```

На каждой отдельной странице мета-теги можно переопределить. Пример для страницы "О нас":

**Файл:** `src/app/about/page.tsx`

```tsx
// строки ~4–11
export const metadata = {
  title:       'О нас',               // ← заголовок вкладки для этой страницы
  description: 'Кто мы такие...',     // ← описание для Google
  openGraph: {
    title:       'О нас — HARUNGI',   // ← заголовок при шаринге в соцсетях
    description: 'Нишевая парфюмерия в Ставрополе...',
  },
};
```
