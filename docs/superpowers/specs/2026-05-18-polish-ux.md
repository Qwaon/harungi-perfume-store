## 2. Page Transitions + Scroll Animations

### PageTransition компонент
- Новый `src/components/PageTransition.tsx` — `'use client'`, обёртка через Framer Motion
- Анимация: `opacity: 0 → 1`, `y: 8 → 0`, duration `0.3s`, ease `[0.22, 1, 0.36, 1]`
- Подключается в `src/app/layout.tsx` вокруг `{children}` внутри `<main>`
- Триггер: смена `pathname` через `usePathname()`

### HowToOrder — stagger на цифрах
- Цифры 01–04 анимируются при появлении в viewport (`whileInView`)
- Stagger: каждая цифра с delay `i * 0.1s`
- Анимация: `opacity: 0, y: 16 → opacity: 1, y: 0`, duration `0.5s`
- `viewport={{ once: true }}`

## 3. ImageGallery + TrustStrip

### ImageGallery
- Стрелки: `rounded-full` вместо текущего без скруглений
- Dot-индикаторы: добавить под главным фото, только если `images.length > 1`
  - Активная точка: pill-shape (`w-4 h-1.5`), неактивная: круг (`w-1.5 h-1.5`)
  - Цвет: активная `bg-ink-900`, неактивная `bg-cream-300`
  - Позиция: `absolute bottom-3 left-1/2 -translate-x-1/2`, flex row, gap-1.5

### TrustStrip
- Заменить dot (`w-1.5 h-1.5 rounded-full bg-gold-500`) на inline SVG-иконки
- Иконки (16×16, stroke, currentColor):
  - "Оригиналы с гарантией" → замок
  - "Распивы от 5 мл" → капля/флакон
  - "Доставка по России" → грузовик
  - "Заказ через Telegram" → telegram-иконка (уже есть в проекте)
