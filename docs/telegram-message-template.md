## Быстрые ответы — подтверждение заказа

**Стандартное подтверждение:**
```
Привет! Заявку получили 🙌
Подтверждаем: [название], [объём], [цена] ₽.
Оплата: [способ]. Как удобно получить?
```

**Если нужна предоплата:**
```
Привет! Всё есть в наличии.
[Объём] [название] — [цена] ₽.
Для брони нужна предоплата [сумма] ₽ на [реквизиты].
Остаток при получении / перед отправкой.
```

**Если нет в наличии:**
```
Привет! К сожалению, [название] сейчас нет.
Могу предложить похожее: [альтернатива 1] или [альтернатива 2].
Или запишу в лист ожидания — как только появится, напишу.
```

## Быстрые ответы — частые вопросы

**"Это точно оригинал?"**
```
Да, только оригиналы. Распивы делаем из оригинальных флаконов.
Могу показать фото флакона или батч-код по запросу.
```

**"Как долго идёт доставка?"**
```
СДЭК или Почта России — обычно 3–7 дней.
Ставрополь — самовывоз или курьер, в этот или следующий день.
```

**"Как оплатить?"**
```
Переводом на карту (Сбер/Тинькофф) или СБП.
Оплата после подтверждения заказа.
```

**"Можно ли вернуть?"**
```
Распивы и открытые флаконы не принимаем обратно — это особенность товара.
Но если есть проблема с заказом, разберёмся в любом случае — напишите.
```

**"Не знаю что выбрать"**
```
Расскажите, что уже нравится носить или какой эффект хотите —
теплее/свежее, на каждый день или на вечер, мужское/женское.
Подберу варианты.
```
