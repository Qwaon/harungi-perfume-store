Нижняя панель навигации (мобиль)

Что должно быть в нижней панели? Выбери вариант структуры.

A

Минимум: 3 пункта

[ 🏠 Главная ] [ 🗂 Каталог ] [ 🛒 Корзина ]

Только самое важное. Никаких лишних вкладок. Максимально чисто.

B

Стандарт: 4 пункта

[ 🏠 Главная ] [ 🗂 Каталог ] [ 🔍 Поиск ] [ 🛒 Корзина ]

Поиск вынесен отдельно — удобно когда знаешь что ищешь.

C

С акцентом на заказ: 4 пункта

[ 🏠 Главная ] [ 🗂 Каталог ] [ 🛒 Корзина ] [ 💬 Написать ]

Кнопка "Написать" (Telegram) всегда доступна — для тех кто хочет спросить или заказать напрямую.

Важно

Сейчас нижней панели нет вообще — только хедер сверху. Добавление bottom nav — это ключевое изменение мобильного UX.
